Basic Maven Commands
---------------------------------------------
1. mvn clean

	Deletes the target/ directory (removes old build files).

	mvn clean

2. mvn compile

	Compiles the source code of the project.

	mvn compile

3. mvn test

	Runs unit tests using a testing framework (like JUnit).

	mvn test

4. mvn package

	Compiles code, runs tests, and packages the project (JAR/WAR).

	mvn package

5. mvn install

	Packages the project and installs it into the local repository.

	mvn install

6. mvn deploy

	Deploys the packaged project to a remote repository.

	mvn deploy

Project Setup Commands
7. mvn archetype:generate

	Creates a new Maven project from a template.

	mvn archetype:generate

8. mvn validate

	Checks whether the project structure is correct.

	mvn validate

Dependency & Info Commands
9. mvn dependency:tree

	Displays the project dependency tree.

	mvn dependency:tree

10. mvn dependency:list

Lists all project dependencies.

mvn dependency:list

11. mvn help:effective-pom

	Shows the final POM after inheritance and defaults.

	mvn help:effective-pom

Clean + Build Combined
12. mvn clean install

	Cleans old build files and installs the new build.

	mvn clean install

What is Maven?

Apache Maven is a build automation and project management tool, mainly used for Java projects.

It helps developers to:

Build projects (compile, test, package, deploy)

Manage dependencies (JAR files and libraries)

Maintain a standard project structure

Automate the build lifecycle

Maven uses a configuration file called pom.xml (POM = Project Object Model) where:

Project details are defined

Dependencies are listed

Plugins are configured

Example:

	mvn clean install


This single command cleans, builds, tests, and installs the project.

Key Features of Maven

	a) Dependency management (automatic download from repositories)

	b) Standard directory structure

	c) Lifecycle-based build system

	d) Plugin-based architecture

	e) Strong community support
